Справка по использованию приложения Медиа-плеер

Добро пожаловать на нашу веб-страницу! Здесь вы найдете необходимую информацию для использования сервиса, желаем приятного использования нашего продукта!

Как начать:

Для начала использования, выполните следующие шаги:
  1. Откройте необходимый вам файл видео или музыку
  2. После загрузки справа будет загружен или загружены несколько файлов
  3. Кликните два раза по вашему файлу для запуска исходного файла
  4. Наслаждайтесь прослушиванием / просмотром вашего медиа-контента

Основные компоненты используемые в программе:

    Файл и его содержимое состоящие из:
    1. Открыть аудио предназначенное для открытия разного Аудио-контента
    2. Открыть видео предназначена для открытия разного Видео-контента
    3. Темы которое содержит дополнительно "Создать" для создания темы пользователем и так же "Вернуть к исходной" если пользователю не понравился интерфейс
    4. Минимализм - убирает лишний функционал для более удобного просмотра, к примеру видео
    5. Выход как ни странно закрывает программу, но она запросит точно ли вы хотите выйти с программы

О программе:

  1. Открывает файл, который показывает информацию о приложении

Справка

  1. Открывает этот документ:)

Панель управления и его содержащие кнопки:

  1. Играть - воспроизводить ваш файл
  2. Пауза - приостанавливает воспроизведение
  3. Стоп - полностью останавливает воспроизведение
  4. Громкость - позволяет переключать методом перемещения громкость воспроизведения
